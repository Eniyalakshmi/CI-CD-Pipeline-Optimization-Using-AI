# CI/CD Pipeline Optimization Using AI 🚀

## 📌 Project Description
This project demonstrates a simple CI/CD pipeline using Jenkins.
It automates build, testing, and execution of a Python project.

## ⚙️ Tools Used
- Jenkins
- GitHub
- Python

## 📂 Project Structure
- Jenkinsfile → CI/CD pipeline script
- main.py → Python execution file
- requirements.txt → Dependencies

## 🚀 How It Works
1. Code pushed to GitHub
2. Jenkins detects changes
3. Pipeline runs automatically
4. Python script executes

## ✅ Output
The pipeline prints success messages in Jenkins console output.
